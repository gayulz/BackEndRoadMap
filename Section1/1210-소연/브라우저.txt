웹브라우저는 우리가 인터넷과 상호 작용하는 방식에서 필수적인 역할
우리가 쇼핑하고,배우고,소통하고,즐겁게 지낼 수 있도록 해주는 광대한 온라인 세계로 가는 관문


<웹과 브라우저의 차이점>
-웹 : 월드 와이드 웹(world wide web)
	인터넷 상에서 서로 연결된 수많은 문서,이미지,비디오,음악,애플리케이션 등이 있는 정보의 체계적인 시스템
	웹은 인터넷을 통해 접근 가능한 정보들의 모음체이며
	사용자들이 정보를 공유하고 검색하며 상호작용할 수 있는 플랫폼
-브라우저 : 웹을 탐색하고 웹 페이지를 표시하는 소프트웨어
	 사용자와 웹 사이의 상호작용을 도와주는 역할
	 사용자가 인터넷을 탐색하고 정보를 얻을 수 있는 창구 역할
-인터넷 : 전 세계적으로 컴퓨터 네트워크들이 상호 연결된 거대한 네트워크
 	정보를 교환하고 전송하고 기술적인 인프라의 집합
	전 서계의 컴퓨터 및 기기들이 서로 통신할 수 있는 기반

=> 웹 = 인터넷 상의 정보들을 모아놓은 전체적인 체계
    브라우저 = 그 정보들을 사용자에게 표시하고 상호작용할 수 있도록 도와주는 소프트웨어
    인터넷 = 컴퓨터와 네트워크들의 전체적인 연결된 구조

<브라우저>
웹페이지,이미지,비디오 등의 콘텐츠를 수신,전송 및 표현하는 소프트웨어
html,css,js,텍스트,이미지 등 웹 리소스를 받아 사용자에게 보여주는 그래픽 사용자 인터페이스(GUI)기반 소프트웨어
=> 웹을 보여주는 프로그램

사용자가 웹사이트를 방문할 때 브라우저 가 필요하다.
브라우저의 주소 표시 줄에 웹사이트의 url을 입력하고 검색하면 브라우저는 웹사이트의 파일이 저장된 서버에 요청을 보내고
이 통신은 http/https 와 같은 프로토콜을 통해 발생
사용자의 요청에 응답하기 위해 서버는 웹사이트의 파일을 다시 보낸다.
이 파일은 HTML,CSS,JS와 같은 언어로 작성되는 경우가 많은데 이러한 코드를 해석하여 사용자가 보는 웹페이지에 렌더링하는 것이
브라우저의 역할
=> 사용자와 서버 사이의 다리 역할
     사용자의 요청을 서버에 보내고 서버의 응답을 사용자에게 쉽게 상호 작용할 수 있는 형식으로 변환

<브라우저 기능>
------------------------------------------------
1. 웹페이지 렌더링
:  사용자가 웹 사이트를 방문하면 서버에서 웹사이트의 HTML,CSS,JS파일을 검색한 후 코드를 해석하고 처리하는 렌더링과정을 거쳐
   웹페이지를 구성해주는 역할
2. 탐색
: 인터넷 탐색
  웹사이트의 주소(URL), 웹 페이지 내 하이퍼링크 를 통해 페이지를 이동
  자주 방문하는 웹사이트 저장하고 구성할 수 있는 북마크 지원
3. 탭 브라우징
: 단일 브라우저 창 내에서 여러 웹페이지를 열 수 있도록 하는 기능
  멀티태스킹을 용이하게 하여 화면을 정리해주고 다른 웹사이트 간 전환이 쉽게 해주는 기능
4. 북마크 및 기록
: 즐겨찾는 웹사이트를 북마크로 저장하는 기능
  북마크 ~ url을 기억하지 않고 빠르게 웹사이트 방문 가능
  웹사이트의 기록을 유지하여 다시 방문하기 위해 검색할 수 있는 연대순 기록 제공
5. 검색 기능
: google, bing 과 같은 인기검색엔진과 통합하여 선택한 검색엔진에 검색어를 보내고 관련 검색 결과 표시
